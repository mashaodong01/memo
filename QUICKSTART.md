# 快速开始

## 🚀 立即使用

### 1. 加载扩展到 Chrome

已经构建完成！按以下步骤加载：

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的**"开发者模式"**
4. 点击**"加载已解压的扩展程序"**
5. 选择 `/Users/mashaodong/Desktop/hytech/mycode/memo/dist` 目录
6. 完成！

### 2. 使用扩展

加载后，点击 Chrome 工具栏中的扩展图标，侧边栏会自动打开。

## ⚠️ 图标说明

当前使用的是临时 SVG 图标。如需正式图标，请：

1. 准备 3 个 PNG 图标文件：
   - 16x16 像素
   - 48x48 像素
   - 128x128 像素

2. 将它们命名为：
   - `icon-16.png`
   - `icon-48.png`
   - `icon-128.png`

3. 放到 `public/icons/` 目录

4. 重新构建：`npm run build`
